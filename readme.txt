== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin files to the `/wp-content/plugins/inline-css-block-gutenberg` directory, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Install Gutenberg plugin if not integrated in WordPress already.
1. Search 'Style' from blocks.
1. Add your CSS code.
